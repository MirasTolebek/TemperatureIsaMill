# 🌡️ TemperatureIsaMill

**Мобильное веб-приложение для быстрого ввода температурных замеров на корпусе мельницы IsaMill**

Инструмент для операторов смены: вводишь цифры — приложение само расставляет запятые и пробелы, затем одной кнопкой копирует отчёт в буфер обмена и отправляет его в Google Таблицу. Никаких данных на сервере не хранится — всё работает прямо в браузере.

---

## 🎯 Зачем это нужно

При ручном вводе температур вида `58,5` оператор вынужден каждый раз переключаться на запятую и вручную ставить пробел между значениями. Это приложение убирает эту рутину:

- Вводишь только цифры: `585` → автоматически форматируется в `58,5`
- После третьей цифры фокус **сам переходит** на следующую строку
- Итоговый текст одним нажатием **копируется в буфер** и **уходит в Google Forms**

---

## 📱 Интерфейс

Приложение состоит из трёх зон:

**Шапка** — переключатель между двумя сторонами мельницы и кнопка очистки.

**Список замеров** — 16 строк для ввода температур. Активная строка подсвечивается синим. При фокусе строка автоматически скроллится в видимую область (важно при открытой клавиатуре на мобильном).

**Кнопка «Копировать»** — формирует отчёт, копирует его в буфер обмена и отправляет данные в Google Таблицу через скрытую форму.

---

## ⌨️ Управление вводом

| Действие | Результат |
|---|---|
| Ввод 3 цифр (напр. `585`) | Авто-форматирование в `58,5`, переход на следующую строку |
| `Backspace` в пустом поле | Возврат на предыдущую строку с удалением последнего символа |
| Кнопка 🗑️ | Запрос подтверждения, затем очистка всех 16 строк текущей вкладки |

Поле принимает только цифры (`inputmode="decimal"`, тип `tel`) — лишние символы отфильтровываются автоматически.

---

## 📋 Формат итогового отчёта

После нажатия «Копировать» в буфер попадает текст вида:

```
Подъемная сторона [27.06.2026 08:45]
1) 58,5
2) 61,2
3) 57,0
...
16) 60,8
```

Если строка не заполнена — ставится прочерк (`-`).

---

## ☁️ Отправка в Google Таблицу

Данные отправляются через **Google Forms** в фоновом режиме (скрытый `<iframe>`), без перезагрузки страницы. В форму передаются два поля:

| Поле | Содержимое |
|---|---|
| Сторона | `Подъемная сторона` / `Неподъемная сторона` |
| Данные | 16 строк замеров |

> ⚠️ ID формы и полей (`entry.*`) зашиты в конфиге `GOOGLE_CONFIG` в начале скрипта — при смене формы нужно обновить только этот блок.

## 🚀 Использование

Приложение — это один файл `index.html`. Открывается в любом браузере, в том числе на телефоне:

**Вариант 1 — GitHub Pages**
1. Форкнуть репозиторий
2. Включить GitHub Pages (`Settings → Pages → Branch: main`)
3. Открыть полученную ссылку на телефоне

**Вариант 2 — локально**
```
Скачать index.html → открыть в браузере
```

**Вариант 3 — любой хостинг / корпоративный сервер**  
Положить `index.html` в корень — и готово.

---

## 🛠️ Настройка под свою форму

Чтобы данные шли в вашу Google Таблицу, нужно:

1. Создать Google Form с двумя полями (тип «Абзац»)
2. Открыть форму → `Inspect → Network` → сделать тестовую отправку → скопировать `action` URL и имена полей `entry.XXXXXXXXX`
3. Вставить в `GOOGLE_CONFIG` в `index.html`:

```javascript
const GOOGLE_CONFIG = {
  action: 'https://docs.google.com/forms/d/e/YOUR_FORM_ID/formResponse',
  sideId: 'YOUR_SIDE_FIELD_ID',
  dataId: 'YOUR_DATA_FIELD_ID'
};
```

## 🔒 Конфиденциальность

- Данные **не хранятся** нигде локально (нет `localStorage`, нет cookies)
- При закрытии вкладки всё сбрасывается
- Единственная внешняя отправка — в Google Forms по вашей собственной форме
