# TemperatureIsaMill
Сайт для быстрого заполнения тепературных данных на корпусе мельницы IsaMill. Данные не хранятся на сервере, и не передается на сервер. Мини веб приложение только для того, чтобы в ручную не печатать точку и не нажимать пробел. Точка или запятая и пробелы ставятся автоматический.
